LSC PCIeAPI Library Documentation Directory
============================================

The documentation is built here, using the Doxygen utility.
The resulting HTML documentation is placed in the html/
directory.  The ReferenceManual.html file can be used to link to
the resulting index.html file.  Index.html is the main
page of the documentation.

The various html files located in the SupportPages directory contain the
extra pages of info related to building, theory and layout of
the source code.

Build the documentation using:
	makedocs.bat
